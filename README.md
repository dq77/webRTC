## webRTC文档

项目代码启动方式

```
node server.js
```

浏览器打开 http://localhost:3000/ 即可查看Demo



Demo包含最多8人的视频会议室，可部署至公网用手机访问。

请确保电脑端使用最新版谷歌浏览器，移动端设备使用最新版微信自带浏览器即可。
